# AEM 6.5 pronto per il mondo - integrazione della traduzione e best practice{#aem-ready-for-the-world-translation-integration-best-practices}

Questa sessione presenta le attuali funzionalità di traduzione AEM e le best practice per la gestione di siti web multilingue. Scopri come strutturare i siti web globali e utilizzare copie per lingua, Live Copy e MSM. La sessione illustra inoltre come ottenere una traduzione dei contenuti più semplice, veloce e gestibile utilizzando funzioni avanzate.

**Presentato da:**

* Ankur Sial, Computer Scientist, Adobe
* Harpreet Neelu, Program Manager, Adobe
